## рҹҢҹ What Is This?

A collection of **50 mini front-end projects** built using only HTML, CSS, and vanilla JavaScript.


## рҹ§  Core Concepts You'll Learn

### рҹ“Ң DOM Manipulation Flow

```
JavaScript вҶ’ selects вҶ’ HTML Elements
     в”Ӯ                      в”Ӯ
     в””в”Җв”Җ reads/writes в”Җв”Җв”Җв–¶  в”Ӯ
                            в–ј
                    Updates the Page
```

### рҹ“Ң CSS Animation Lifecycle

```
Initial State  в”Җв”Җв–¶  Trigger (hover/class/scroll)
                         в”Ӯ
                         в–ј
               Transition / Keyframe plays
                         в”Ӯ
                         в–ј
               Final State reached
                         в”Ӯ
               (optionally reverses)
```

### рҹ“Ң Fetch API (used in API projects)

```
JS: fetch(URL)
       в”Ӯ
       в–ј
 HTTP Request sent to API server
       в”Ӯ
       в–ј
 Server returns JSON response
       в”Ӯ
       в–ј
 .json() parses the data
       в”Ӯ
       в–ј
 DOM is updated with results
```
